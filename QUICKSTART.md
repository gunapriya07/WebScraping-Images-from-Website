# Quick Start Guide

## Web Image Scraper - ProjectsHUB

### Instant Setup (3 steps)

1. **Install Dependencies**
   ```bash
   pip install -r requirements.txt
   ```

2. **Launch Application**
   ```bash
   python main.py
   ```

3. **Start Scraping**
   - Enter website URL
   - Click "Start Scraping"
   - Find images in downloads folder

### Command Line Usage
```bash
python main.py --cli -u https://example.com
```
